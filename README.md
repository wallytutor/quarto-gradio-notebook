# Quarto-Gradio Notebook

A notebook editor conceived on Gradio for editing Quarto notebooks.

## Features

- [ ] File navigation tree fixed on the left side.
- [ ] Document sectioning with multi-level titles.
- [ ] Dynamic markdown cell creation with LaTeX support.
- [ ] Dynamic Python cell creation and execution.
- [ ] Code/text and section collapse for focus.
- [ ] Virtual environment selection and kernel creation.
- [ ] A simple variable explorer for checking state.
- [ ] Portable thanks to Electron interface.
